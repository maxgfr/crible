# Financial Market Data APIs
Posted on Apr 25, 2026
All figures and plan details below are as of 2026-04 (unless stated otherwise).
## Alpha Vantage
Free API key with lifetime access; free tier includes 25 requests/day : Alpha Vantage Support
Covers equities, FX, crypto, and technical indicators for prototypes and automations.
Official MCP server is referenced for LLM and agent integrations.
## Finnhub
Free plan includes 60 API calls/minute (personal use): Finnhub pricing
Useful for company fundamentals, news, and market data (limits vary by endpoint).
## Twelve Data
Basic plan is free with 800/day API credits and 8 calls/min : Twelve Data pricing
Covers stocks, forex, crypto, and technical indicators (limits apply).
## Financial Modeling Prep (FMP)
Basic tier is free with 250 calls/day : FMP pricing
Strong for fundamentals, ratios, statements, and screening workflows.
## marketstack
Free plan includes 100 requests/month with EOD data and 1-year history: marketstack pricing
Useful for low-volume dashboards and daily-close analytics.
## CoinGecko API
Crypto market data API with a free getting-started option (no credit card): CoinGecko API
Useful for crypto prices, market caps, exchange data, and related endpoints.
## Stooq
Free historical OHLCV downloads via CSV: Stooq
Useful for backtesting and research when EOD data is enough.
No official API guarantees.
