## The Problem
I wanted to find a list of stock market ticker symbols that is well maintained with regular updates so that my application could consider new IPOs and drop the rare company which goes private. It is easy to find a webpage that shows off ticker symbols. However, all the pages I found required the user to click through hundreds of pages.
It’s OK to click through hundreds of pages as a human user, but an algorithm needs raw data. It’s possible to write a web scraper to painstakingly get information one page at a time from any given finance website, but that is not a very efficient solution.
Take this page from Nasdaq’s website for example. You’ll find a screener which lists all the stocks available on the Nasdaq, New York Stock Exchange, and the American Stock Exchange. The page lists 8,443 stocks total at the time of writing.
The problem is — this webpage only displays 25 symbols at a time. The rest are hidden behind AJAX calls. I don’t want to write a web scraper that has to surf through 337 different pages.
You may have noticed the page does have a “ Download CSV ” file button. That download button works great if you’re a manual user in the browser. However, you’ll notice that they obfuscated the link that it downloads the file from if you try to write a script to complete the task.
I bet you could figure out how they generate the link if you poked around enough. Or, you could write something up using Selenium to push the button. Regardless, I wanted a simpler solution, and I found it.
## The Solution
NasdaqTrader.com is a website that looks like it hasn’t been updated since the early 2000s. They don’t even use HTTPS on the main site. But they do provide legitimate trading products, market data, and news.
NasdaqTrader.com does one more thing that I love: they host a public FTP server with market information. I haven’t seen it advertised anywhere other than in the comments of a StackExchange post . Later, I found more information on their site here .
You can login to the server at ftp.nasdaqtrader.com . Use the username, “ anonymous , ” and a blank password . You’ll want to look specifically in the SymbolDirectory folder. There are two files you should be interested in downloading inside this directory: nasdaqlisted.txt and otherlisted.txt .
Both files follow a CSV style format, but they use “|” as a delimiter.
## Automation
It’s quite easy to write a Bash or PowerShell script to complete this task for you. However, those solutions aren’t very portable. I opted for a Python script instead.
Python’s ftplib module is really easy to use. Just create an FTP object, change directories to SymbolDirectory with the .cwd() method, and download both files with .retrlines(). I’ve provided a stripped down version of the script I use down below.
import ftplib
import os ftp = ftplib.FTP(“ftp.nasdaqtrader.com”)
ftp.login(“anonymous”, “”)
ftp.cwd(“SymbolDirectory”) files_to_download = [“nasdaqlisted.txt”, “otherlisted.txt”] for file in files_to_download:
with open(file, “wb”) as f:
ftp.retrbinary(f”RETR {file}”, f.write) ftp.quit()
