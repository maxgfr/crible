### What Ghostfolio does best
1 Self-host an AGPL-3.0 open-source portfolio and personal finance dashboard, or use the managed Ghostfolio Premium cloud.
2 Track net worth across cash, stocks, ETFs, cryptocurrencies, custom assets, accounts, and platforms.
3 Manage portfolio activity with transaction create/read/update/delete workflows, imports, exports, charts, static risk analysis, and standard performance timeframes.
4 Use the progressive web app and Android wrapper for mobile-first access to the same Ghostfolio instance.

### What OpenBB does best

## Questions we keep getting
What's the difference between Ghostfolio and OpenBB? Ghostfolio leans toward portfolio, data visualizations, and APIs & data feeds, while OpenBB puts more weight on APIs & data feeds, sheets / excel add-ins, and data visualizations. They overlap in 2 categories, so for most people it comes down to workflow preference and price.
How much do Ghostfolio and OpenBB cost? Good news: both Ghostfolio and OpenBB have free plans, so you can run them side by side and only pay if you hit a wall.
Which is better for beginners: Ghostfolio or OpenBB? Ghostfolio is the friendlier place to start; its interface takes less getting used to. Both work fine once you're past the basics.
Do Ghostfolio and OpenBB have APIs? Yes, both offer API access, so developers and quants can pull data programmatically or wire up their own integrations.
Should I choose Ghostfolio or OpenBB? It depends on what you're after. Pick Ghostfolio if portfolio matter to you; go with OpenBB if you'd rather have sheets / excel add-ins and custom dashboards. And if you only need the basics both share, let price decide.
